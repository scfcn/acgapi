使用方法

简单的API调用方式，快速集成到您的项目中

自动识别设备类型

访问以下路径，API会自动根据设备类型返回对应的图片资源：

https://acg.qxzhan.cn/get

示例代码（JavaScript）：

// 使用fetch API获取图片
fetch('https://acg.qxzhan.cn/get')
  .then(response => {
    // 获取重定向后的图片URL
    const imageUrl = response.url;
    // 显示图片
    document.getElementById('image').src = imageUrl;
  })
  .catch(error => console.error('Error:', error));
